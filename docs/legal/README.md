# Legal Pack (Context Lattice)

Effective date: 2026-02-18

This directory is the current legal and licensing package for Context Lattice.
It is written for practical launch readiness and should be reviewed by counsel
before broad commercial rollout.

## Included documents

- `COMMERCIAL_LICENSE.md` - commercial use terms and BSL interaction.
- `TERMS_OF_SERVICE.md` - hosted service terms.
- `PRIVACY_POLICY.md` - data handling and user rights.
- `DPA.md` - data processing addendum baseline.
- `ACCEPTABLE_USE_POLICY.md` - prohibited use and abuse controls.
- `SUBPROCESSORS.md` - third-party processor inventory.

## License model summary

- Source license: Business Source License 1.1 (`LICENSE` in repo root).
- Additional use grant: local/internal usage as defined in `LICENSE`.
- Commercial license required for usage outside grant terms.

## Review cadence

- Operational/legal review at each public release.
- Subprocessor list update whenever a new hosted dependency is added.
- Privacy/terms update when product data flows materially change.

## Disclaimer

This package is not legal advice. Final legal approval should come from your
attorney for your jurisdictions and deployment model.

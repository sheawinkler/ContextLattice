# ContextLattice v1.0.0

Public launch release for ContextLattice as an MCP memory service.

## Highlights

- Local-first MCP memory service with private-by-default security posture.
- Durable write fanout with queue/backpressure controls.
- Federated retrieval across Qdrant, Mongo raw, MindsDB, Letta, and memory-bank.
- Learning rerank loop for retrieval quality improvements over time.
- Launch-ready docs bundle: architecture, installation, integrations, troubleshooting, updates.

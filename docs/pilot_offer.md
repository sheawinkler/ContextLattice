# Paid Pilot Offer (2026-01-24)

## Title
**Context Cost Audit + ContextLattice Pilot (2–4 weeks)**

## Objective
Quantify baseline token spend and demonstrate cost + reliability gains from memory.

## Deliverables
- Baseline metrics: token spend, long-context rate, truncation/failure rate.
- ContextLattice deployment (self-host or hosted).
- After metrics + ROI report (anonymized if requested).

## Requirements from partner
- Share baseline token usage (or allow measurement during pilot).
- One workflow to optimize (e.g., agent tasks, research assistants, IDE bots).
- Read-only access to logs or metrics (anonymized acceptable).

## Timeline
- Week 1: baseline capture + integration
- Week 2–3: optimization + monitoring
- Week 4: ROI summary + rollout recommendation

## Pricing posture
- Low fee to remove friction (e.g., $500–$2k pilot) + optional credits.

## Success criteria
- 30–80% reduction in prompt tokens on target workflow.
- Lower failure rate and improved completion.

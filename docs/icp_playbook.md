# ICP Playbook (2026-01-24)

## ICP A: LLM ops / platform teams (fastest revenue)
- **Pain:** high long-context spend + truncation failures + agent instability.
- **Value:** predictable memory spend; fewer failures; measurable savings.
- **Offer:** paid pilot + ROI report.
- **Sales motion:** warm intro → technical validation → 2–4 week pilot.
- **Success criteria:** % reduction in prompt tokens; improved task success rate.

## ICP B: Vibecoders / IDE agent builders (growth)
- **Pain:** context limits break flow; manual copy/paste; lost decisions.
- **Value:** persistent memory, less rework, faster agent loops.
- **Offer:** free OSS + cheap hosted tier.
- **Motion:** self-serve docs + community channels.
- **Success criteria:** daily active usage; retention over 7/30 days.

## ICP C: Enterprise AI platform teams (high ACV)
- **Pain:** data privacy, auditability, compliance, vendor risk.
- **Value:** private context management with controls (BYOK, private networking).
- **Offer:** enterprise pilot with compliance bundle.
- **Motion:** procurement-friendly security docs + POC.
- **Success criteria:** security review passed; paid contract signed.

## Shared messaging
- Tagline: Fix the context problem plaguing vibecoders everywhere.
- Promise: Private, enterprise-grade context management for teams and agents.
- Near-free entry: OSS core; pay only when scale/controls needed.

## CTA copy (finalized)
- Primary CTA: Start local-first in minutes.
- Secondary CTA: Run a 2-4 week ROI pilot.
- Buyer CTA: Book an enterprise architecture review.
